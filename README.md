# 🚀 RAPD Algorithm  
### (RAPD Optimization Algorithm)  
**Robust Adaptive Perceived-Detail Driven Multi-Metric Optimization Algorithm**

---

## 📌 Overview

Image enhancement is a fundamental task in image processing that aims to improve visual quality while preserving essential structural and perceptual information. Classical enhancement techniques such as Histogram Equalization (HE) and Contrast Limited Adaptive Histogram Equalization (CLAHE) are widely used to improve contrast; however, they often suffer from limitations including over-enhancement, noise amplification, and distortion of natural brightness.

To overcome these challenges, this project introduces the **Rapid Algorithm (RAPD Optimization Algorithm)** — a novel **multi-metric optimization-based image enhancement framework**. RAPD is specifically designed to achieve **balanced, natural, and detail-preserving enhancement** by jointly optimizing multiple perceptual and statistical image quality parameters within a unified adaptive optimization framework.

---

## 🎯 Key Idea

Most existing image enhancement techniques focus on optimizing a **single image quality metric**, which frequently results in degradation of other important visual characteristics. In contrast, RAPD formulates image enhancement as a **multi-metric optimization problem**, where several quality measures are optimized simultaneously. This ensures that improvements in one metric do not compromise others, leading to visually consistent and robust enhancement.

---

## 🔍 Image Quality Metrics Optimized

The RAPD algorithm jointly optimizes the following image quality metrics:

- **PSNR (Peak Signal-to-Noise Ratio)**  
  Controls noise amplification and minimizes distortion in the enhanced image

- **SSIM (Structural Similarity Index)**  
  Preserves structural integrity and perceptual similarity with the original image

- **Entropy**  
  Enhances information richness and improves visibility of fine details

- **Brightness Variance (BV)**  
  Maintains natural illumination and prevents unnatural brightness shifts

- **Contrast Measures**  
  Improves visibility and distinguishability of important image regions
